# HTTP/Web Protocol Checklist

- [ ] Methods validated (GET/POST/PUT/DELETE)
- [ ] Correct status codes (200/201/400/401/403/404/500)
- [ ] Required headers present (Content-Type, Authorization)
- [ ] CORS behavior validated
- [ ] Cache headers reviewed
- [ ] Cookie/session/token behavior validated
- [ ] TLS/HTTPS only on production endpoints
